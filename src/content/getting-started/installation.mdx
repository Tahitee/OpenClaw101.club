# 安装与部署

本篇将带你完成 OpenClaw 的安装和初始部署。无论你使用的是 Linux 服务器、macOS 还是 Windows（通过 WSL），都可以按照本教程快速上手。

## 系统要求

OpenClaw 对运行环境的要求并不高。只需要一台能联网的机器，安装好 Node.js 环境，即可开始。我们会详细介绍每个步骤。

## 安装步骤

从获取安装脚本到首次运行 Gateway，整个过程通常只需要几分钟。本节会覆盖常见的安装方式和可能遇到的问题。

## 验证安装

安装完成后，我们会通过几个简单的命令来验证 OpenClaw 是否正确安装并可以正常运行。
