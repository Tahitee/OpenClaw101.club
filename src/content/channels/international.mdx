# WhatsApp / Telegram / Discord 接入

OpenClaw 原生支持 WhatsApp、Telegram 和 Discord 三大国际主流通讯平台。本篇将分别介绍每个平台的接入方式和注意事项。

## WhatsApp 接入

通过 WhatsApp Business API 实现接入。我们会介绍如何申请 API 权限、配置 Webhook、处理消息格式等。

## Telegram 接入

Telegram Bot 是最容易上手的接入方式之一。从创建 Bot 到配置 OpenClaw，只需几步即可完成。

## Discord 接入

将 OpenClaw 作为 Discord Bot 接入你的服务器，让 AI 助手参与群组对话。支持斜杠命令、消息回复等多种交互方式。
